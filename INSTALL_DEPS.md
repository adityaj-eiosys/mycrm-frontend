# Install Missing Dependencies

Run this command in the `mycrm-frontend` directory:

```bash
cd mycrm-frontend
yarn install
```

This will install all the missing Radix UI packages:
- @radix-ui/react-label
- @radix-ui/react-select
- @radix-ui/react-slot
- @radix-ui/react-dialog
- @radix-ui/react-dropdown-menu

After installation, restart the dev server:

```bash
yarn dev
```
